angularjs_exercise
==================

1. Create directive with 3 UI tabs.
2. Navigate between the tabs (state) using UI-ROUTER.
3. Create 3 jsons files Which contain different information each (first name,last name, address, phone number).
3.1. in one of the jsons create number of objects and Display theme in the UI using ng-repeat.
4. Create SERVICE which will call JSON for using $http.
5. Display the answer received from the JSON in the VIEW.
6. Create Drop Down (filter) under which the page can be used to sort any field.
