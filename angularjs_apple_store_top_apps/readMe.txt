Apple Store Top Apps Report
 
Goal
Create a single page report showing the current top 12 paid apps in the Apple App Store.   
 
Specifications
* For each app display app name, icon, price, company/author, release date, average user 
	rating, rating count (number of ratings) 
* Allow sorting the apps by relevant properties 
* Clicking on an app should display app description and screenshots 
 
Notes
* You may use any language, packages, tools and platforms you prefer 
* References you might find use useful: 
* https://rss.itunes.apple.com/us/  
* https://www.apple.com/itunes/affiliates/resources/documentation/itunes­store­web­
